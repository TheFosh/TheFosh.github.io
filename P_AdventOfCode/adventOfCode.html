Personal project

Day 1

Description

Given a text file, find the first and last numbers in the string. In part one, only the actual digits (i.e. 1, 2, 3, …) counts. For part two, any word that spells a digit (i.e. one, two, three, …) also counts. Find the first and last valid digit of a line of a string. These digits form a two-digit number. Do this for all lines in the string and add the numbers found together to get the answer to the problem.

Day 2

Description

The input given for this problem represents games of pulling cubes out of a bag. Each line is a different game of pulling cubes and has a varying out of times pulled from it (with replacement). For part one, determine which games could be possible to have 12 red, 13 green, and 14 blue. This will lead to the answer. For part two, determine the smallest possible number of cubes in each game’s bag. This will lead to a new answer.

Day 3

Description

Given an input resembling a 2D gird of numbers and symbols, solve part one and two. For part one, locate all numbers that are next two a symbol that isn’t the “.” symbol. This will lead to the answer for this part. For part two, find all “*” symbols that have exactly two numbers next to them. These numbers will lead to part two’s answer.

Day 4

Description

The input given represents scratch cards where there are winning numbers and your numbers on either side of the cards. For part one, determine the number of pairs for winning numbers and your numbers on cards. This number is used for calculating the score which is the answer to this part. For part two, every time you find a winning card, that card makes copies of the cards after based on the number of winning pairs on it. These new copies of cards are also able to generate copies of cards after it as well. Count the number of original and copied cards that contain a winning pair. This is the answer for part two.
