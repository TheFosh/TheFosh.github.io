Day 1

Description

Given a text file, find the first and last numbers in the string. In part one, only the actual digits (i.e. 1, 2, 3, …) counts. For part two, any word that spells a digit (i.e. one, two, three, …) also counts. Find the first and last valid digit of a line of a string. These digits form a two-digit number. Do this for all lines in the string and add the numbers found together to get the answer to the problem.

Highlighted Code

intInventory::getLastNum(string _line_) { int firstNum = -1; char tmpChar; for (int i = index; i < _line_.size(); ++i) { // Checks if current char is in the range 0-9. // If yes, it converts it to an int. tmpChar = _line_[i]; if (tmpChar >= '0' && tmpChar <= '9') { firstNum = tmpChar - '0'; } // PART 2 for (int w = 0; w < wordSize; ++w) { if (_line_.rfind(numWords[w]) == i) { firstNum = w; } } } return firstNum; }

This code was used for finding the last char possible from the given string line. It makes use of the fact that you can perform mathematical expressions and comparisons with chars. This lets me figure out if which char is an actual number and which isn’t.

Possible Improvments

  • With specifically the function for last char, I could have started at the end instead of after the first char found. That would likely make this faster.

Day 2

Description

The input given for this problem represents games of pulling cubes out of a bag. Each line is a different game of pulling cubes and has a varying out of times pulled from it (with replacement). For part one, determine which games could be possible to have 12 red, 13 green, and 14 blue. This will lead to the answer. For part two, determine the smallest possible number of cubes in each game’s bag. This will lead to a new answer.

Highlighted Code

const Game Game::operator=(const Game& rhs) { if (this != &rhs) { delete[] red; delete[] blue; delete[] green; arraySize = rhs.arraySize; red = new int[rhs.arraySize]; for (int i = 0; i < rhs.arraySize; ++i) { red[i] = rhs.red[i]; } blue = new int[rhs.arraySize]; for (int i = 0; i < rhs.arraySize; ++i) { blue[i] = rhs.blue[i]; } green = new int[rhs.arraySize]; for (int i = 0; i < rhs.arraySize; ++i) { green[i] = rhs.green[i]; } } return *this; }

I was very pleased when this bit of code worked. It made my use of pointers at arrays a lot simpler.

Day 4

Description

The input given represents scratch cards where there are winning numbers and your numbers on either side of the cards. For part one, determine the number of pairs for winning numbers and your numbers on cards. This number is used for calculating the score which is the answer to this part. For part two, every time you find a winning card, that card makes copies of the cards after based on the number of winning pairs on it. These new copies of cards are also able to generate copies of cards after it as well. Count the number of original and copied cards that contain a winning pair. This is the answer for part two.

Highlighted Code

int Inventory::recursiveCounting(list<Card>::iterator itr, int total) { int count = (*itr).countPairs();// Count of current card // Goes through each iterator after the current one based on the count above for (int i = 0; i < count; ++i) { ++itr; // Sets the total count to the number of times this method is recursevly called // This in effect counts the number of copies of a card total = recursiveCounting(itr, ++total); } return total; }
